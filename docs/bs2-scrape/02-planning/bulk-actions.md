# Module 02: Planning — bulk-acties

**Gescraped op**: 2026-05-13 (batch 1)

## Batch 1 bevindingen

**Geen bulk-actie-mechaniek zichtbaar** in batch 1:
- Geen checkbox-headers in grid
- Geen "Selecteer alles" knop
- Geen bulk-dropdown bovenaan
- Per dienst-cell geen checkbox

## Mogelijke bulk-acties (te testen in batch 2)

1. **Drag-and-drop bulk-move**: meerdere diensten naar andere dag/cel
2. **Rechts-klik context-menu** op cell met bulk-opties
3. **Genereren wizard** = effectief bulk-create van diensten via template (vermoedelijk N diensten per generation)
4. **Optimaliseren wizard** = effectief bulk-update van toewijzingen (alle openstaande in periode)

## BS1-implicatie

Indien batch 2 geen rij-checkboxes vindt → geen bulk-acties nodig in BS1 Planning.
Genereren + Optimaliseren als bulk-equivalenten implementeren.
