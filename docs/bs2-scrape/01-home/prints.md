# Module 01: Home — print / PDF / export

**Gescraped op**: 2026-05-13

## Conclusie: GEEN print/PDF/export

Op de Home-pagina geen print-, PDF-, Excel- of CSV-export functionaliteit:
- Geen "Print" of "Exporteren" knop in toolbar
- Geen download-icoon op nieuws-cards
- Geen "Print all news" knop
- Notification-overview `/notifications` heeft ook geen export-knop

## BS1-implicatie

Voor Module 01 Home: **geen print/PDF te implementeren in Fase E**.

## Hoe vastgesteld

- Scan van Home page: geen printer/download iconen
- Notification-bell dropdown: alleen "Alles bekijken" knop
- `/notifications` overview: alleen tabs + lijst, geen toolbar-export
- Nieuws-detail modal: alleen close-X, geen "Print artikel"
