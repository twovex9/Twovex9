# Module 21 — Facturen importeren — BEHAVIORS

## Step 1: Bestand kiezen
1. Klik upload-area of sleep bestand
2. File-input accepts: SVG/PNG/Excel/CSV/JPG/JPEG/PDF/.docx/.doc
3. Klik Vergroten → preview vergroot
4. Klik Ander bestand → vervang upload
5. Klik X → clear upload
6. Klik Volgende (`#fi-next-1`) → naar step 2

## Step 2: Controleren
1. Preview van geüpload bestand
2. Klik Vorige (`#fi-back-2`) → terug naar step 1
3. Klik Importeren (`#fi-next-2`) → start import process
4. Bij succes: bestand toegevoegd aan import-history-tabel

## Import history (BS1 extra)
- Tabel: Bestandsnaam / Type / Grootte / Geïmporteerd op / Acties
- Default empty state: "Nog geen bestanden geïmporteerd"

## Naar facturen
- Klik link → navigeer naar facturen-overzicht
